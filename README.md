## 開発サーバー

```
npm install
npm run dev
```

開発サーバーは常に `http://localhost:1398/`（`strictPort`）で待ち受けます。既定の Vite ポート(5173)では動かないため、トラブルシューティングや他チームへの共有の際は上記 URL を参照してください。
